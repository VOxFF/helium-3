Detailed Description

Abstract base class for all mechanical units in the Helium-3 simulation.

Examples of concrete subclasses: • MiningTruck — mobile excavator that travels, mines, unloads
• UnloadStation — fixed platform that queues trucks and transfers ore
• SolarCharger — static device that recharges batteries, logs uptime

Detailed Description

Abstract base for immobile facilities that service trucks.

Stations should also participate in the global update loop (i.e., are energised machines with state/logging).

Concrete subclasses (e.g., UnloadStation, future ChargeStation) decide what to do with the queued trucks: unload payload, recharge batteries, perform maintenance, and so on.
